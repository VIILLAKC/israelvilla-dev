Graficando datos de voz con Plotly

En este proyecto utilicé Django junto con Plotly para visualizar datos de voz. Plotly proporciona herramientas muy útiles para crear gráficos interactivos en la web.

Notificaciones de detección de rostros con Django Channels

En este proyecto utilicé Django Channels para enviar notificaciones en tiempo real cuando se detecta un rostro en una imagen. Utilicé una red neuronal convolucional (CNN) entrenada para detectar rostros y luego envié una notificación a través de Channels cuando se detectó un rostro en una imagen.
